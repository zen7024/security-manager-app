# セキュリティ管理アプリ

ブラウザだけで動く、軽量なセキュリティ管理アプリです。
単一HTMLファイルで構成されており、インストール不要で利用できます。

## できること

- IT資産の登録と重要度管理
- 脆弱性の記録、深刻度、対応状況の管理
- インシデントの記録とリスク値の可視化
- 監査チェック項目の進捗管理
- マスターパスワードで暗号化する個人用パスワードボルト
- JSONでのバックアップと復元

## iPhoneで使うには

今の `file://` 起動のままでは、外出先のiPhoneからは使えません。
外で使うには、このHTML一式を `https://` で公開する必要があります。

おすすめの流れ:

1. Netlify か GitHub Pages にこのフォルダを公開する
2. iPhone の Safari で公開URLを開く
3. 共有メニューから `ホーム画面に追加` を実行する
4. 念のため `JSONを書き出し` で定期バックアップする

このアプリは PWA 向けの `manifest.webmanifest` と `sw.js` を追加済みです。
そのため、公開後は iPhone でもアプリに近い形で使いやすくなります。

## 保存先

データはブラウザの `localStorage` に保存されます。
別のブラウザや別端末とは共有されません。

パスワード管理タブの内容は、保存時に `PBKDF2 + AES-GCM` で暗号化されます。
マスターパスワードを忘れると復号できません。
